jQuery Plugin - contentfader Demo

Demo of the jQuery Plugin - contentfader.

The plugin is executed with the default parameters

$.fn.contentfader.defaults = {
    items: 3 // max items to show
   ,itemWidth: 320
   ,crossfade: true // true | false
   ,easing: 'swing' // swing | linear
   ,speed: 3000 // in milliseconds
   ,interval: 8000 // in milliseconds
 }

For more information, please visit the project page jQuery Plugin - contentfader
